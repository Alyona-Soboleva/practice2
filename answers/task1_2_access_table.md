# Задание 1.2 — Таблица модификаторов доступа

## Теоретическое напоминание

В Java существуют четыре уровня доступа:

| Модификатор | Свой класс | Тот же пакет | Наследник (другой пакет) | Весь мир |
|-------------|:----------:|:------------:|:------------------------:|:--------:|
| `private`   |     ✅     |      ❌      |           ❌             |    ❌    |
| *(без модификатора)* |  ✅  |   ✅   |           ❌             |    ❌    |
| `protected` |     ✅     |      ✅      |           ✅             |    ❌    |
| `public`    |     ✅     |      ✅      |           ✅             |    ✅    |

**Важно для этого задания:** класс `Employee` находится в пакете `company.core`, а класс `HRSystem` — в пакете `company.app`. Это **разные** пакеты. `HRSystem` **не** наследует `Employee`.

## Ваша таблица

Для каждой строки (A–H) из `HRSystem.java` укажите:

| Строка | Обращаемый член                  | Модификатор | Компилируется? (да/нет) | Причина                                                           |
|--------|----------------------------------|-------------|:-----------------------:|-------------------------------------------------------------------|
| A      | `emp.name`                       |             |           да            | public — доступен из любого пакета                                |
| B      | `emp.age`                        |             |           нет           | protected — недоступен из другого пакета без наследования         |
| C      | `emp.salary`                     |             |           нет           | package-private (без модификатора) — недоступен из другого пакета |
| D      | `emp.password`                   |             |           нет           | private — доступен только внутри класса Employee                  |
| E      | `emp.getRole()`                  |             |           да            | public — доступен из любого пакета                                |
| F      | `emp.promote(5000)`              |             |           нет           | protected — недоступен из другого пакета без наследования         |
| G      | `emp.printSummary()`             |             |           нет           | package-private — недоступен из другого пакета                    |
| H      | `emp.validatePassword("secret")` |             |           нет           | private — доступен только внутри класса Employee                  |
